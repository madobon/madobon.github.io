# チートシート

## 構造化のテスト

```java
@RunWith(Enclosed.class)
public class EnclosedTest {
    public static class Test1 {
        @Before
        public void before() {
            // 前処理
        }
        @Test
        public void testCase() {
            // テスト
        }
        @After
        public void after() {
            // 後処理
        }
    }
    public static class Test2 {
        @Before
        public void before() {
            // 前処理
        }
        @Test
        public void testCase() {
            // テスト
        }
        @After
        public void after() {
            // 後処理
        }
    }
}
```

## 例外のテスト

```java
@Test(expected=Exception.class)
public void testCase() {
  try {
    hoge()

  } catch (Exception e) {
    assertEquals("xxxx is not specified.", e.getMessage)
    throw e;
  }
}
```
