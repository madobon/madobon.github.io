#### 見出し４

```js
  var func = function(x, y, z) { return x + y + z }

  func.bind(this)(1, 2, 3) // 6 値の束縛をしない関数を返却し、引数(1, 2, 3) を与える。
  func.bind(this, 1)(2, 3) // 6 引数(1)を束縛した関数を返却し、引数(2, 3) を与える。
  func.bind(this, 1, 2)(3) // 6 引数(1, 2)を束縛した関数を返却し、引数(3) を与える。
```
